## Category Heuristics

- Names containing `visual`, `image`, `a-plus`, `copy`, `main-image` go under 图片与 A+ 内容.
- Names containing `keyword`, `dropdown`, `listing-keyword` go under 关键词与运营辅助.
- Names containing `research`, `product-research`, `market` go under 市场调研与选品.
- Names containing `ingredient`, `compliance`, `formula`, `rebuild`, `boundary`, `supplement-safe` go under 成分研发、合规与 IP 风险.
- Names containing `index`, `router`, `skill` and used to manage other skills go under Skill 管理与路由.
