## 特有字段

### 品牌 → 创作者/IP
| 字段 | 说明 | 来源 |
|------|------|------|
| niche | 垂直领域 (1-3个标签) | Bio / Content analysis |
| platform_primary | 主力平台 | 粉丝分布 |
| followers_total | 总粉丝（跨平台合计） | 各平台公开数据 |
| engagement_rate_avg | 平均互动率 | 采样计算 |
| monetization_streams | 变现渠道数及构成 | 内容分析 / 公开披露 |
| estimated_monthly_income | 月收入估算 | *估算，基于可见变现信号* |
| content_frequency | 更新频率 (posts/week) | 统计 |
| content_format_primary | 主要形式 (图文/短视频/中视频/直播/podcast/audio) | |
| team_size | 团队规模 (solo / 1-3助手 / full studio) | about页 / 推测 |
| mcn_signed | 是否签约MCN | 披露 / 行业消息 |

### 痛点（创作者特有）
| 维度 | 痛点 |
|------|------|
| 算法 | algorithm change(算法变动导致流量暴跌) / reach decline(触达下降) / shadowban(限流) |
| 心理 | creative burnout(创作倦怠) / imposter syndrome(冒充者综合征) / comparison trap(攀比陷阱) |
| 收入 | income instability(收入不稳定) / payment delays(结算延迟) / platform fee changes(平台抽成变化) |
| 平台 | platform risk(封号/政策变更) / lack of ownership(没有所有权) / algorithm opacity(算法不透明) |
| 运营 | content saturation(内容饱和) / comment moderation(评论审核) | copyright issues(版权问题) |

### 内容分析（额外字段）
| 字段 | 说明 |
|------|------|
| viral_coefficient | 传播系数（分享率 × 每次分享平均曝光） |
| save_rate | 收藏率（尤其小红书风格平台） |
| share_rate | 转发率 |
| comment_sentiment | 评论情感倾向（positive/negative/mixed/controversial） |
| content_lifecycle | 内容生命周期（爆发期多长 / 长尾流量持续多久） |
