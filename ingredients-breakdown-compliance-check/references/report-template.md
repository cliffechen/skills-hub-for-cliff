# 报告模板

除非用户要求不同的输出格式，否则使用本结构。

## 1. 执行摘要

包含：
- 审查的目标
- 整体风险状况
- 最大的已确认风险
- 最大的未解决问题

## 2. 输入快照

列出：
- 输入类型
- 销售平台和区域
- 审查的资料
- 使用的工具

## 3. 成分拆解

使用紧凑的表格。

| 成分或术语 | 分类 | 重要性说明 | 置信度 |
| --- | --- | --- | --- |
| 名称 | 通用 / 品牌 / 专有 / 不明 | 一句话原因 | 弱 / 中等 / 强 |

## 4. 知识产权风险发现

回答：
- 哪些术语看起来是商标或品牌化的
- 哪些成分看起来受供应商控制或专有的
- 哪些项目产生了专利或 FTO 问题
- 缺少哪些采购或授权证据

## 5. 合规与市场发现

回答：
- 成分状态或新颖性顾虑
- FDA 宣称或标签顾虑
- 亚马逊受限产品或文件要求顾虑
- 可能的后续证据请求

## 6. 风险评分卡

| 领域 | 评级 | 置信度 | 主要原因 |
| --- | --- | --- | --- |
| 商标或品牌成分 | 低 / 中 / 高 | 弱 / 中等 / 强 | 简短原因 |
| 专利或 FTO | 低 / 中 / 高 | 弱 / 中等 / 强 | 简短原因 |
| FDA 或标签合规 | 低 / 中 / 高 | 弱 / 中等 / 强 | 简短原因 |
| 亚马逊市场风险 | 低 / 中 / 高 | 弱 / 中等 / 强 | 简短原因 |

## 7. 下一步检查

以最有价值的 3-5 项后续检查结束，例如：
- 所有者或供应商授权证明
- 最终标签图片
- USPTO 记录审查
- 专利族审查
- 成品 COA
- 亚马逊 TIC 状态
