# 提示词库

每次新调查开始时使用本文件。选取最接近的起始模板，然后根据用户的实际输入进行调整，继续完整的工作流。

## 通用触发词

- `用 $ingredients-breakdown-compliance-check 分析这个 ASIN，拆解所有可能的成分，并标记商标、专有成分、专利、FDA 和亚马逊美国站风险。`
- `用 $ingredients-breakdown-compliance-check 调研这个品牌，告诉我哪些是通用成分、哪些是品牌成分、哪些可能产生侵权或合规风险。`
- `用 $ingredients-breakdown-compliance-check 审查这个产品配方，识别 TM/R 成分、专有成分和亚马逊美国站 Listing 风险。`

## ASIN 导向触发词

- `用 $ingredients-breakdown-compliance-check 分析 ASIN B09ZQ74495，进行成分拆解、商标成分风险、专利关联成分风险和亚马逊美国站合规暴露分析。`
- `用 $ingredients-breakdown-compliance-check 反向拆解这个 ASIN 的成分堆栈，告诉我哪些可以安全模仿，哪些应该避免。`
- `用 $ingredients-breakdown-compliance-check 调查这个亚马逊竞品 ASIN，并通过 AnySearch 和 Sorftime 交叉验证其成分信息。`

## 品牌导向触发词

- `用 $ingredients-breakdown-compliance-check 分析 AlgaeCal 品牌，梳理其主要产品中使用的所有品牌化、专有或有风险的成分。`
- `用 $ingredients-breakdown-compliance-check 调查这个保健品品牌，找出反复出现的成分体系，并标记商标、专利和 FDA/亚马逊风险。`
- `用 $ingredients-breakdown-compliance-check 拆解这个品牌的核心产品，告诉我它们的成分名称是通用的还是受保护的。`

## 单一成分触发词

- `用 $ingredients-breakdown-compliance-check 分析 AHCC，告诉我它是通用成分、商标成分还是专有成分体系。`
- `用 $ingredients-breakdown-compliance-check 调查 Mesophyllum superpositum 的标签安全性、商标风险、专利风险和亚马逊美国站保健品风险。`
- `用 $ingredients-breakdown-compliance-check 审查这个成分名称，告诉我能否安全地写在 Supplement Facts 或营销文案中。`

## 配方或标签触发词

- `用 $ingredients-breakdown-compliance-check 审查这个 Supplement Facts 成分表，识别所有成分类别的知识产权和亚马逊美国站合规风险。`
- `用 $ingredients-breakdown-compliance-check 将这个配方拆解为通用成分和品牌成分，告诉我上架前需要准备哪些文件。`
- `用 $ingredients-breakdown-compliance-check 评估这个草稿标签是否可能触发 FDA 标签问题或亚马逊受限产品审查。`

## 竞品仿制触发词

- `用 $ingredients-breakdown-compliance-check 分析这个竞品，告诉我配方中哪些部分在商业上模仿更安全，哪些看起来是专有或受保护的。`
- `用 $ingredients-breakdown-compliance-check 找出这个竞品中的高风险成分，并建议在构建类似产品前需要进一步调研什么。`
- `用 $ingredients-breakdown-compliance-check 将这个产品中的品牌护城河与通用配方区分开来。`

## 中文高频触发词

- `用 $ingredients-breakdown-compliance-check 分析这个 ASIN，把所有可能的成分拆出来，并标记商标、专利、FDA、亚马逊美国站风险。`
- `用 $ingredients-breakdown-compliance-check 调研这个品牌，区分哪些是通用成分，哪些是商标成分，哪些可能有专利或专有原料风险。`
- `用 $ingredients-breakdown-compliance-check 检查这个 Supplement Facts，告诉我哪些成分写法安全，哪些会有侵权或合规风险。`
- `用 $ingredients-breakdown-compliance-check 调查这个成分名能不能写在美国站标签里，并交叉验证站内外信息。`
- `用 $ingredients-breakdown-compliance-check 分析这个竞品，看看哪些成分可以模仿，哪些是 branded ingredient 或 proprietary ingredient。`
