# The Ordinary 品牌风格指南（膳食补充剂品牌对标）

本文件定义了品牌对标的文案风格、视觉语言和表达原则。所有 Listing 文案、A+ Content、图片文案必须遵循此指南。

## 品牌核心定位

做美国膳食补充剂类目的 "The Ordinary"——以成分透明、临床实证、极简美学打破保健品行业的信息不对称。

品牌 Tagline 理念：**Clinical Formulations with Integrity.**（临床配方，恪守诚信）

## 品牌人格

| 维度 | 我们是 | 我们不是 |
|------|-------|---------|
| **透明度** | 成分、浓度、机制全部公开，不藏着掖着 | 用"专有配方""神秘成分"忽悠消费者 |
| **科学性** | 每个声称都有临床数据或文献支撑 | 靠玄学、传统、"古方"来背书 |
| **克制** | 不夸大功效，不贩卖焦虑，用事实说话 | 用恐惧营销、夸大before/after效果 |
| **可及性** | 专业内容用普通人能懂的方式表达 | 故作高深、堆砌术语让普通消费者望而却步 |

## 文案写作原则

### 1. 成分先行（Ingredient-First）

**核心规则**：产品名称和文案都以核心成分及其浓度/规格开头，让消费者第一时间知道自己在买什么。

The Ordinary 范例：
- "Saccharomyces Ferment 30% Milky Toner" — 产品名直接包含成分+浓度+剂型
- "Hyaluronic Acid 2% + B5 — A Hydration Support Formula with Hyaluronic Acid, Pro-Vitamin B5, and Ceramides"

**补充剂品牌适配**：
- 标题示例："Magnesium Glycinate 400mg — Highly Bioavailable Magnesium for Muscle and Nerve Support"
- 而非："Amazing Relaxation Formula — Feel Calm Tonight!"

### 2. 反推销语气（Anti-Sales Voice）

**核心规则**：文案读起来像一个有知识的朋友在客观介绍产品，而不是销售员在推销。

具体表现：
- 用 "helps support" 而非 "amazing results"
- 用 "designed to" 而非 "guaranteed to"
- 用 "may contribute to" 而非 "will transform"
- 用 "supports" / "promotes" / "contributes to" 替代 "cures" / "treats" / "fixes"
- 避免感叹号堆砌和 ALL CAPS 强调

**禁用表达清单**：

| 禁止使用 | 替代方案 |
|---------|---------|
| Amazing / Incredible / Miracle | 删除，用数据替代 |
| Guaranteed results | Clinically studied / Research-supported |
| Best / #1 / Most effective | 删除绝对化用语 |
| Transform your life / life-changing | Supports / promotes / contributes to |
| 100% safe / No side effects | Well-tolerated in clinical studies |
| Cures / Treats / Heals / Prevents disease | Supports / Maintains / Promotes (structure/function) |
| Buy now / Don't miss out | 删除紧迫感话术 |
| Doctor approved / Doctor recommended | 除非有真实背书，否则删除 |

### 3. 教育优先（Educate, Don't Sell）

**核心规则**：文案的首要目的是帮助消费者理解成分和作用机制，购买决定是教育的自然结果。

The Ordinary 的 A+ 内容范例：
- 用科学图解解释成分如何工作（如角质代谢过程图）
- 对比同类产品帮助消费者做选择（如 "What Toner Should I Use?" 对比图）
- 展示成分搭配方案（如 "Build Your Regimen: PREP → TREAT → SEAL"）

**补充剂品牌适配**：
- 用图解说明营养成分在体内的作用路径
- 对比不同形态的成分（如 "Magnesium Glycinate vs. Magnesium Citrate — Which Is Right for You?"）
- 展示搭配方案（如 "Morning Stack: Vitamin D3 + K2 + Magnesium"）

### 4. 极简美学在文案中的体现

**核心规则**：每一个词都要有存在的理由，删掉一切不增加信息的修饰词。

The Ordinary 文案特征：
- 短句为主，一句一个信息点
- 不用形容词堆砌（"luxurious, silky, revolutionary"全部删除）
- 大量留白，信息密度低但信息价值高
- 双语标注（英文/法文）增加国际感和专业感

**补充剂品牌文案的句式范例**：

```
Key ingredients:

**Magnesium Glycinate (400mg)**
A highly bioavailable form of magnesium. Chelated to glycine for enhanced absorption and gentle digestion.

**Vitamin D3 (5000 IU)**
Supports immune function, bone health, and calcium absorption. Sourced from lanolin.

**Zinc Picolinate (15mg)**
An easily absorbed form of zinc. Supports immune defense and antioxidant activity.
```

注意：
- 每个成分单独一段
- 成分名 + 浓度 → 一句话说清楚是什么 → 一句话说清楚它做什么
- 没有多余修饰，没有情感渲染

### 5. Before/After 和临床数据的表达规范

The Ordinary 的做法：
- Before/After 必须标注临床测试条件："Clinical testing on 34 panellists using the product 2x/day for 4 weeks."
- 数据声称必须可溯源："Source: Circana/U.S. Prestige Beauty Total Measured Market, 12 Months Ending December 2023"
- 功效表述用 "promotes" / "helps" / 而非 "will give you"

**补充剂品牌适配**：
- 所有临床数据声称必须标注来源和测试条件
- Before/After 在补充剂中尤其敏感——必须符合 FDA 规定，不能暗示治疗疾病
- 使用 "supports" 类结构函数声称，而非疾病声称

## 视觉与文案的配合规则

### 主图文案
- 纯白背景，产品居中
- 标签上只出现：品牌名 + 成分名 + 浓度 + 一句话描述
- 无促销文字、无星级评价、无"Best Seller"标签叠加

### A+ Content 图文模块类型

| 模块类型 | 用途 | The Ordinary 范例 |
|---------|------|-----------------|
| **成分解读** | 解释核心成分是什么、怎么工作 | "Key ingredients:" + 成分列表 + 功能说明 |
| **功效对比** | 展示使用前后或不同产品对比 | "Promotes supple skin with improved elasticity" + Before/After |
| **使用指南** | 展示产品搭配方案或使用步骤 | "Build Your Regimen: PREP → TREAT → SEAL" |
| **选择指南** | 帮消费者在多个产品间做选择 | "What Toner Should I Use?" 对比图 |
| **机制科普** | 图解成分的作用机理 | "How Saccharomyces Ferment exfoliates" 三步图解 |
| **市场地位** | 展示品牌/产品的市场认可 | "#1 Serum Brand in the US" + 数据来源 |

### 图片文案排版规则
- 字体：干净无衬线体（The Ordinary 用 Raleway），全大写或正常大小混用
- 颜色：黑色文字 + 白色/浅灰背景，不使用彩色文字
- 层级：大标题（模块主题）→ 正文（成分/功效描述）→ 脚注（数据来源）
- 图文关系：文字与产品图/图解之间用线条或留白区隔，不使用箭头或花哨指示

## Listing 各区域的文案风格

### Product Title（产品标题）
格式：`[品牌名] [核心成分] [浓度/规格] — [一句话功能描述]`

范例：
- "The Ordinary. Hyaluronic Acid 2% + B5, A Hydration Support Formula with Hyaluronic Acid, Pro-Vitamin B5, and Ceramides"
- 补充剂适配："[Brand] Magnesium Glycinate 400mg — Highly Bioavailable Magnesium for Muscle, Nerve, and Sleep Support, 120 Capsules"

### Bullet Points（五点描述）
每条遵循：**功能主题 + 一句话功效说明**

范例：
- "Immediate Plumping Hydration"
- "Instantly Smoother-Feeling Skin"
- "Promotes Tighter-Feeling Skin"
- "Lightweight Texture"

补充剂适配：
- "Highly Bioavailable Magnesium — Chelated to glycine for superior absorption and gentle digestion"
- "Supports Restful Sleep — Promotes relaxation and healthy sleep patterns naturally"
- "Muscle and Nerve Support — Contributes to normal muscle function and nerve signaling"

### A+ Content 正文
遵循教育优先原则，每个模块回答一个消费者关心的问题。语言风格：专业但平易近人，像在和一个好奇的朋友解释科学知识。

### Product Description
简短概括产品定位 + 核心成分 + 适用人群。不堆砌关键词，不重复 bullet points 已有内容。
