# 实验室对接清单

这个清单的目标不是教工厂怎么生产，而是让实验室第一轮快速判断：

- 这个方案能不能做
- 哪些点要先确认
- 哪些文件必须补

## 一、产品身份

- 产品概念名
- 目标市场和渠道
- 目标剂型：capsule / caplet / tablet / softgel / gummy / powder / liquid
- Serving size
- Servings per container
- 目标规格和包装形式

## 二、活性成分核对

| 项目 | 实验室需要确认什么 |
| --- | --- |
| 成分名称 | 标签名、化学/营养形式、植物部位等是否准确 |
| 建议标示量 | 每份、每粒是否可实现 |
| 原料规格 | assay、标准化、效价、流动性等是否合适 |
| 法规/标签 | DV、`Daily Value not established`、警示语是否需要 |
| 相容性 | 水分、pH、氧化、热、压片、包衣敏感性 |
| 安全边界 | 上限、交互、敏感人群、特殊警示 |

## 三、辅料与工艺目标

这里写目标，不写制造参数。

- 目标递送方式：普通胶囊、肠溶、缓释、延释等
- 辅料避免项：sugar alcohol free、titanium dioxide free、gelatin free、allergen free 等
- 体验目标：无异味、无明显后味、易吞咽
- 稳定性目标：防氧化、防潮、防热、防光，适合泡罩还是瓶装
- 尺寸约束：单粒大小、片重、单份粒数

## 四、实验室需要准备或确认的文件

- 原料 CoA
- 成品 CoA
- 身份和效价检测方案
- 重金属
- 微生物
- 过敏原声明
- 如适用：残留溶剂
- 稳定性测试计划
- 如适用：溶出或肠溶释放测试
- 标签量 overage 建议
- 活性成分与辅料的相容性说明

## 五、开放问题

最终文档里建议总是给实验室留下 3-5 个最关键问题，例如：

- 目标活性量能否装进当前设定的单位大小？
- 当前原料形式是否是这个剂型里最稳的？
- 在 clean label 目标下，哪些辅料是必不可少的？
- 肠溶或稳定性叙事需要什么测试才能站得住？
- 在打样前，最需要先锁定的是哪一个技术风险？
