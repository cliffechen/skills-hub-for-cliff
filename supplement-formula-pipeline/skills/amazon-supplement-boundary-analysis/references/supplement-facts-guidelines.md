# Supplement Facts Guidelines

## Default Package

Use a 90-count package by default:

```text
90 capsules or softgels per bottle
```

Serving math:

```text
Servings Per Container = 90 / Serving Size
```

Preferred serving sizes:

- 1 capsule/softgel: 90 servings.
- 2 capsules/softgels: 45 servings.
- 3 capsules/softgels: 30 servings.

Avoid 4 capsules/softgels by default because 90 does not divide cleanly.

## Default Other Ingredients

Use this template unless the user provides another:

```text
Distilled Water, Maltitol Syrup, Maltitol Powder, Isomalt, Pectin, Citric Acid, Sodium Citrate, Natural Flavor, Natural Color, Carnauba Wax.
```

## Drafting Rules

- Output Supplement Facts as Markdown with code blocks for label-like views.
- Include `Daily Value not established` for ingredients without DV.
- State that the formula is exploratory and needs label/legal/manufacturing review.
- Keep nutritional roles distinct; do not make every ingredient claim the same benefit.
- Avoid disease claims, anti-aging reversal claims, and branded-comparison claims.
